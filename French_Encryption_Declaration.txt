### **Question 1: What are your app's main features and intended purpose?**

**Grid** is a **location-based social networking app** designed for proximity-based connections and real-time messaging. The app serves as a modern, privacy-focused alternative to traditional dating and social apps by emphasizing **authentic, real-time connections with people in your immediate vicinity**.

#### **Core Features:**

1. **📍 Location-Based User Discovery**
   - Real-time grid display of active users within proximity
   - Dynamic geographic filtering showing people nearby
   - Location-based matching using CoreLocation services
   - Users appear on a visual grid representing their physical proximity

2. **💬 Instant Messaging & Chat**
   - Real-time one-on-one messaging with nearby users
   - CloudKit-powered messaging with push notifications
   - End-to-end encryption mode for privacy-conscious users
   - Message read receipts and delivery confirmation
   - Background messaging with APNs integration

3. **👤 Profile Management**
   - Sign in with Apple for secure authentication
   - Rich user profiles with multiple photos (up to 10)
   - Bio and personal information sharing
   - Device-specific profiles allowing multiple devices per Apple ID

4. **🔒 Privacy & Safety Features**
   - User blocking and reporting system
   - Content moderation using NaturalLanguage framework
   - App Tracking Transparency compliance
   - Privacy-first design with optional location sharing
   - Built-in abuse prevention and safety tools

5. **⭐ Social Interaction Features**
   - Star/favorite users for easy reconnection
   - Profile overlay for detailed user viewing
   - Filter options (show starred users only)
   - Interactive grid interface with haptic feedback

6. **🛡️ Advanced Security**
   - Optional end-to-end encryption for sensitive conversations
   - CryptoKit-based encryption profiles
   - Secure key exchange and management
   - Privacy mode for enhanced security

#### **Technical Implementation:**
- Built entirely on **Apple's native ecosystem** (CloudKit, CoreLocation, SwiftUI)
- **100% free infrastructure** using Apple's services
- Real-time synchronization across devices
- Offline capability with local data persistence
